Presentación

Soy Jose Herrera, y esta es mi primera página web. En en el curso PWDC de Egg Cooperation estoy aprendiendo desarrollo web para crear sitios web visualmente atrapantes y funcionales.

Lo que se viene...

A lo largo de este curso, estaré aprendiendo:

HyperText Markup Language
«Lenguaje de marcado de hipertexto», hace referencia al lenguaje de marcado para la elaboración de páginas web.

Cascading Style Sheets
«Hojas de estilo en cascada», es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito HTML

JavaScript
es un lenguaje de programación interpretado. Se define como orientado a objetos, basado en prototipos, imperativo, débilmente tipado y dinámico.

Estos lenguajes me ayudarán a crear una base sólida en el desarrollo web front-end.
